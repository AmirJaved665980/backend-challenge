# Backend-Challenge
- Serverless node image uploader api

## Pre-requisite
- AWS cloud account
- Node installed

## Get Started 
```
Installation:
$ npm install -g serverless
$ serverless --version
$ serverless config credentials --provider aws --key <your_access_key_id> --secret <your_access_key_secret>
```

```
Hello World:
$ serverless create --template hello-world
$ serverless deploy
```
